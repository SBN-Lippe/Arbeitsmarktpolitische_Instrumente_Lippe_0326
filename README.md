# Arbeitsmarktpolitische Instrumente Dashboard

Interaktives Dashboard für die Analyse von arbeitsmarktpolitischen Instrumenten im Kreis Lippe.

## 🚀 Features

- Interaktive Filter nach Berichtsmonat und Instrumententyp
- Kennzahlen für Teilnehmende und Neueinsteiger
- Visualisierungen: Linien- und Balkendiagramme
- Detaillierte Datenübersicht in Tabellenform

pip install -r requirements.txt
